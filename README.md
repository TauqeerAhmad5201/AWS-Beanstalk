# AWS-Beanstalk

AWS Elastic Beanstalk is a cloud computing service offered by Amazon Web Services (AWS) that allows developers to deploy and manage web applications and services with ease. 

AWS Elastic Beanstalk is an AWS-managed service for web applications. Elastic Beanstalk is a pre-configured  directly take up your application code and environment configurations and use it to automatically provision and deploy the required resources within AWS to run the web application.
